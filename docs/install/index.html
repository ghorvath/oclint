Installation

Installation of M.O. is pretty easy, download archived executable binary and libraries, and unzip them into a folder. That’s it.

At this stage of development, M.O. is only pre-compiled for 64-bit Intel Mac OS X Lion/Snow Leopard users. For Linux users, please read how to compile to build M.O. yourself for now.

After get a copy, you can invoke M.O. in command line terminal/console.
