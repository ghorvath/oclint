How to Select Rules

All rules should be loaded from OCLint_executable_folder/rules folder.

You can easily drag and drop rules to this folder to select the rules you are going to use for inspections, we call this plug-and-use rule loading mechanism. Check out rules packaged together with OCLint

And you can add your own rules to this folder, for detail, read how to write our own rules.
