Documents

OCLint is a static source code analysis tool for improving quality and reducing defects by inspecting Objective-C/C++/C code. As OCLint is under active development, so part of the documents here may behind the latest version. If you found something out of date and have trouble using OCLint, I’ll be happy to help.

Contributing

OCLint began as a research project at the University of Houston. Since then, OCLint has grown to be a 100% open source project. If you want to help out, fork OCLint on Github and send us a pull request.
